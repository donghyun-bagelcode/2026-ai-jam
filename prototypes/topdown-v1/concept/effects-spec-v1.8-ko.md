## 1. 범위

- 출시 범위: `World 1 Basic(10) + Hard(10)`
- 대상: 2D 파티클, 스프라이트 애니메이션, UI 트랜지션, 상태 피드백 연출
- 제외: 월드2~5 전용 신규 테마 VFX(후속 업데이트)

## 2. 공통 연출 원칙

- 퍼즐 가독성을 최우선으로 하며, 플레이 타일 경계를 가리지 않는다.
- 상태 변화(획득/활성/잠금)는 0.2~0.6초 내 인지 가능해야 한다.
- 연출은 짧고 명확하게 유지하며 입력 반응 지연을 만들지 않는다.
- 저사양 대응을 위해 `Low` 옵션에서 파티클 수 40% 축소 가능해야 한다.
- 색상만으로 구분하지 않고 아이콘/모양 변화와 함께 사용한다.

## 4. 규칙 연동 포인트

- 제자리 스와이프(이동 0칸): 이동 FX를 재생하지 않고 입력 피드백만 허용한다.
- 마지막 키 획득 슬라이드에서 포탈 연속 도달: `FX-17 -> FX-19`를 같은 슬라이드 내 연쇄 재생한다.
- 랜덤 포탈(하드 Stage 3~10): 선택된 후보 포탈만 `FX-17`, `FX-18` 활성화 대상으로 본다.

## 5. 구현 우선순위(권장)

1. 필수 이펙트(`FX-01,02,03,04,06,09,11,13~19,24,27`) 구현
2. 하드 핵심(`FX-20,23`) 구현
3. 권장/선택(`FX-05,07,08,10,12,21,22,25,26,28`) 순차 추가

## 6. QA 체크리스트

1. 키/포탈 상태 변화 시 이펙트가 1회만 재생되는가
2. 슬라이드 중 입력 잠금 상태에서 중복 재생이 없는가
3. 저사양 옵션 적용 시 프레임 드랍 없이 동작하는가
4. 하드 기믹(키순서/일방통행/워프)에서 오작동 없이 트리거되는가
5. 결과 팝업 연출 중 버튼 연타 시 중복 전환이 발생하지 않는가
