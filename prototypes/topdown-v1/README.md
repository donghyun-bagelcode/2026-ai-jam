# topdown-v1

실행 파일은 `execute/` 폴더에 모아두었습니다.

## 처음 실행 시 (macOS)
1. 터미널을 열고 `execute/` 폴더로 이동
2. `bash setup-mac.sh` 실행
3. 이후 `run-mac.command` 또는 `run-game.app`을 더블클릭으로 실행 가능

## 실행 / 종료
- 실행: `execute/run-mac.command` 또는 `execute/run-game.app` 더블클릭
- 종료: `execute/stop-mac.command` 또는 `execute/stop-game.app` 더블클릭

## 수동 실행 (대안)
```bash
cd execute
bash run-mac.command
bash stop-mac.command
```
