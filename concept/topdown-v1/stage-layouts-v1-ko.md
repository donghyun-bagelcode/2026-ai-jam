# 월드별 스테이지 레이아웃 초안 v1 (World 1~5)

- 작성일: 2026-03-04
- 용도: 스테이지 1~10 데이터 초안(기획/구현 공통 참조)
- 검증 상태: 수동 설계 초안 (솔버 검증 전)

## 공통 표기 규칙

- 맵 크기: `9 x 9`
- 좌표계: 좌상단 `(0,0)`, 우하단 `(8,8)`
- 이동 가능 기본 타일: `.`
- 벽: `#`
- 시작 위치: `S`
- 포탈: `P` (World 3 제외)
- 키: `k` (World 2는 `1`,`2`,`3`)

## 검토 메모

- 본 문서는 "설계 초안"으로, 각 스테이지의 실제 `M`값과 클리어 가능성은 엔진 솔버 검증이 필요하다.
- 특히 World 3/5는 상태공간(동적 출구/워프 사용 이력)이 커서 자동 검증 후 미세 조정 권장.
