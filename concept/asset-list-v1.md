# Key Route — 에셋 목록 v1

- 작성일: 2026-03-04
- 기준: game-design-v1-ko.md (World 1)
- 아트 컨셉: 파스텔 컬러, 부드러운 분위기, 2D 탑뷰

---

## 1. 타일 (2종)

| 에셋 | 설명 | 비고 |
|------|------|------|
| 바닥 타일 | 걸을 수 있는 칸. 아이보리/베이지 톤의 부드러운 사각형 | 둥근 모서리, 약간 볼록한 느낌 |
| 벽 타일 | 이동 불가 영역. 어두운 색 또는 빈 공간(구멍) | 바닥과 명확히 구분 |

## 2. 오브젝트 (4종)

| 에셋 | 설명 | 비고 |
|------|------|------|
| 키 | 수집 아이템. 빛나는 구슬/보석 형태 | 스테이지당 3개 |
| 비활성 포탈 | 잠긴/흐릿한 원형 포탈 | 키 3개 수집 전 상태 |
| 활성 포탈 | 빛나는/열린 원형 포탈 | 키 3개 수집 후 상태 |
| 플레이어 | 캐릭터 스프라이트 | 아이들 상태 기본, 방향별 슬라이드는 선택 |

## 3. UI 요소 (8종)

| 에셋 | 설명 | 사용 화면 |
|------|------|----------|
| 별 (빈) | 미획득 별 | 클리어 팝업, 스테이지 선택 |
| 별 (채움) | 획득한 별 | 클리어 팝업, 스테이지 선택 |
| 키 수집 표시 | HUD용 키 아이콘 (0/3 ~ 3/3) | 인게임 HUD |
| 이동 카운터 | 슬라이드 횟수 표시 | 인게임 HUD |
| 재시도 버튼 | 스테이지 재시작 | 클리어 팝업, 인게임 |
| 뒤로가기 버튼 | 스테이지 선택으로 | 인게임, 클리어 팝업 |
| 다음 스테이지 버튼 | 다음 스테이지로 | 클리어 팝업 |
| 잠금 아이콘 | 미해금 스테이지 | 스테이지 선택 |

## 4. 배경 (2종)

| 에셋 | 설명 | 비고 |
|------|------|------|
| 게임 배경 | 인게임 배경. 부드러운 그라디언트 | 파스텔 톤 |
| 선택 화면 배경 | 스테이지 선택 화면 배경 | 월드 느낌 또는 단색 그라디언트 |

## 5. 이펙트 (3~4종)

| 에셋 | 설명 | 구현 방식 |
|------|------|----------|
| 키 획득 이펙트 | 반짝임/파티클 | 코드 또는 스프라이트 시트 |
| 포탈 활성화 이펙트 | 빛이 퍼지는 연출 | 코드 또는 스프라이트 시트 |
| 클리어 이펙트 | 축하 연출 (컨페티 등) | 코드 또는 스프라이트 시트 |
| 슬라이드 잔상 | 이동 궤적 표시 (선택) | 코드 기반 |

## 6. 사운드 (8종)

| 에셋 | 설명 | 비고 |
|------|------|------|
| BGM | 잔잔한 퍼즐 배경음악 | 루프 가능 |
| 슬라이드 | 캐릭터 이동 효과음 | 짧고 부드러운 |
| 키 획득 | 반짝이는 수집 소리 | |
| 포탈 활성화 | 웅~ 하는 활성화 소리 | |
| 클리어 | 짧은 팡파레 | |
| 별 획득 | 딩딩딩 (별 수에 따라) | |
| 버튼 탭 | UI 터치 피드백 | |
| 실패/리셋 | 스테이지 재시작 소리 (선택) | |
